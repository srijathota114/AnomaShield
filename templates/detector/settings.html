Parameter Guide
Z-Score Threshold

Higher values reduce false positives but may miss some outliers. Typical range: 2.0-5.0. Default: 4.0 (very conservative).

IQR Multiplier

Higher values reduce false positives. Typical range: 1.5-3.0. Default: 2.5 (very conservative).

Consensus Threshold

Number of methods that must flag a row for it to be considered poisoned. Higher values = more conservative detection. Range: 1-4. Default: 2.

ML Parameters

Contamination/Nu parameters for machine learning methods. Higher values = more sensitive detection. Should generally increase with dataset size.
